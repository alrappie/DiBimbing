aplikasi_sederhana.ipynb sebagai program utama untuk mendemonstrasikan fungsionalitas dari masing-masing kelas tersebut

workflow pada aplikasi_sederhana.ipynb 
1. melakukan pemanggilan dari class yang telah dibuat dengan melakukan import 
2. melakukan inisiasi pada Order
3. melakukan inisiasi OrderProcessor
4. memasukan semua order pada OrderProcessor
5. kemudian melakukan uji fungsionalitas dengan memanggi fungsi calculate_total_revenue -> untuk menghitung total amount dari setiap order yang ada
6. calculate_total_tax ->  untuk menghitung total tax dari setiap order yang ada
7. display_order -> untuk menampilkan keseluruhan order
